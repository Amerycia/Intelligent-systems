Опис проєкту

Цей проєкт демонструє створення та навчання простої нейронної мережі на Python з використанням бібліотеки PyTorch для класифікації рукописних цифр з набору даних MNIST.
Модель навчається розпізнавати цифри від 0 до 9 та показує високу точність на тестовому наборі.

Запуск проєкту
1. Встановити залежності

У терміналі виконайте:

pip install torch torchvision matplotlib


Підтримуються версії Python 3.10–3.13.

2. Запустити код

Скрипт запускається командою:

python mnist_nn.py

Що робить програма

автоматично завантажує датасет MNIST (при першому запуску),

проводить попередню обробку даних,

будує та навчає нейронну мережу,

виводить приклади зображень із мітками,

показує прогрес навчання,

обчислює точність моделі на тестових даних.

Очікувана точність моделі: приблизно 96–97%.
